## Features:
- Case Flow
  - Saving
  - Loading
  - Full Annotation Support
  - AI Summary
    - Identifies Weaknesses, Strengths, and Holes in the Round's Flow
- Upcoming: Card Cutter

## How to Use:
- Setup .env file in main directory
- Run the Docker-Compose File.
- Go to localhost:8000
- Will also be publicly available on the web soon.
